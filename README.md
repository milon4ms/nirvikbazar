# Nirvik Bazar

## Project Description
Nirvik Bazar is an innovative e-commerce platform designed to connect buyers and sellers efficiently. It offers a seamless shopping experience with various features that cater to both ends of the marketplace.

## Features
- User-friendly interface
- Secure payment options
- Real-time order tracking
- Responsive design for mobile and desktop
- Product reviews and ratings
- Advanced search and filter options

## Technology Stack
- Frontend: React.js
- Backend: Node.js with Express
- Database: MongoDB
- Styling: CSS Modules and SASS
- Deployment: Heroku

## Getting Started
To get started with the Nirvik Bazar project:
1. Clone the repository:
   ```bash
   git clone https://github.com/milon4ms/nirvikbazar.git
   ```
2. Navigate into the project directory:
   ```bash
   cd nirvikbazar
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Run the application:
   ```bash
   npm start
   ```
5. Open your browser and go to `http://localhost:3000` to see the app in action!
